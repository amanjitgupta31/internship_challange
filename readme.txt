1.a) Permutation

create a .csv file with ', ' between two words.

The .jar file is located at "T1\target\T1-0.0.1-SNAPSHOT.jar".

run the 'T1-0.0.1-SNAPSHOT' jar file with a command line argument same as the path of the .csv file.



1.c) Word Suggestion

create a .csv file with ',' between two words.

The .jar file is located at "T\target\T-0.0.1-SNAPSHOT.jar".

run the 'T-0.0.1-SNAPSHOT' jar file with a command line argument same as the path of the .csv file followed by a space an then the input word.
